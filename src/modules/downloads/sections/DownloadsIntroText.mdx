This page provides access to extracts of the datasets presented in the tool. The
extracts are organised by country and data source.

Datasets are prepared individually for each country for download and available
for all users through this page.

Please note that some datasets are currently marked unavailable. If any of these
are of interest, please [raise an issue](https://github.com/nismod/irv-datapkg/issues/new) or contact us.

For programmatic interactions with this service, please also refer to the [API
documentation](https://global.infrastructureresilience.org/extract/redoc),
[Python client](https://github.com/nismod/irv-autopkg-client) and [usage
examples](https://github.com/nismod/irv-autopkg-use).
