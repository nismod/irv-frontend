# Pixel Driller Export (Stub)

This README will describe the structure and meaning of the exported CSV and JSON files.

- Each domain contributes a pair of files: a CSV with data and a JSON file with metadata.
- Filenames follow the pattern `{domain}__{other_keys}.csv` and `.json`.
- The first line of each CSV is a comment describing the columns.
- JSON metadata files currently contain a stub object and will be extended in future versions.
