This page demonstrates **`ArticleMap`** with **built-in** view layers only: two embeds for **Aqueduct** fluvial and coastal flood rasters, each with **primary roads** coloured by **expected annual damage (EAD)** for that hazard (`infrastructureViewLayer` + damage style matching Risk → Infrastructure). Each embed uses a **nested `RecoilRoot`** so interaction state stays separate from the main map.

For a **custom GeoJSON-only** example, see [Article map: custom layer only](/articles/20260329-article-map-custom-layer).
