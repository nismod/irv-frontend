This article demonstrates how to:

- Co-locate small datasets next to an article
- Render them using a local React component based on **Vega-Lite**
- Keep the article itself mostly markdown, embedding only declarative components

## Embedded chart

Below is a bar chart rendered from `data.json` in the same folder via a local `ArticleChart` component:

<ArticleChart
  description="This is a bar chart rendered from `data.json` in the same folder via a local `ArticleChart` component"
  spec={{
    data: { name: 'values' },
    mark: 'bar',
    encoding: {
      x: { field: 'category', type: 'ordinal', axis: { title: 'Category' } },
      y: { field: 'value', type: 'quantitative', axis: { title: 'Value' } },
    },
    width: 400,
    height: 300,
  }}
  data={data}
/>

You can inspect this folder to see how the MDX, data, and component live together:

- `article.mdx` – this file
- `data.json` – small inline dataset
