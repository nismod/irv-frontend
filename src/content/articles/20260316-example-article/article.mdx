# Example narrative article

This is a **sample article** to show how narrative content can live in MDX. You can use normal markdown: headings, lists, links, and so on.

## Structure

- Put the **meta** export at the top (title, optional description and date).
- Write the body in markdown.
- Embed interactive components with JSX where needed.

## Embedded components

Below is a callout box (an example of a custom component available in all articles):

<Callout title="Note">
  This callout is rendered by a React component passed into the MDX. You can add more components

</Callout>

Another paragraph after the callout. Links work with the app router: go to the [data viewer](/view/hazard) or back to the [intro](/).
